# anyTodo

### 使用说明
基于redux和Semantic的管理任务清单例子

#### 项目截图

#### 技术栈
> 详情可参阅 package.json
* React
* Redux
* Webpack
* ES6 + Babel
* Semantic（UI）


#### 本地运行

执行<br/>
<code>git clone https://github.com/viviier/anyTodo.git</code>

安装依赖包<br/>
请使用<code>yarn</code> 或者 <code>npm i</code>

启动服务<br/>
<code>npm run dev</code>

访问地址<br/>
ocalhost:8866

打包编译为静态页面<br/>
<code>npm run buld</code>
